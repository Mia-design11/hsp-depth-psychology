# 高敏感深度心理咨询 - Claude Code Skill

专为高敏感HSP/高共情/易内耗人群设计的专业心理咨询技能。

## ✨ 核心亮点
- **20年资深咨询师视角**：以标准成人心理咨询流程为基底，不强行把所有问题归因为“你就是高敏感”，避免陷入自我标签怪圈。
- **七层深度分析框架**：情绪分层梳理、躯体信号解读、认知偏差拆解、模式溯源，拒绝空泛安慰。
- **高敏感专属保护**：教你建立情绪边界、共情剥离、能量止损，让敏感天赋不再消耗你的生活。
- **禁用否定话术**：全程不说“别想太多”“你太敏感了”，给你安全、无评判的支持空间。

## 🎯 解决的痛点
- 精神内耗、过度复盘、睡前胡思乱想停不下来
- 人际敏感、习惯性讨好、不敢拒绝、怕冲突
- 情绪容易被他人带走、感官/信息过载导致心累
- 完美主义拖延、过度自省、自我价值感偏低
- 产后情绪敏感、激素波动叠加内耗困扰
